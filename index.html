베어링 수명 계산기 — 구현 표준 안내

본 도구의 위치: 예비 설계 / 사이징 / 교육용. 최종 사양서 검증에는 SKF SimPro Expert, MESYS, Romax 등 전문 도구를 권장합니다.

📐 적용 표준

  • L_10 기본 수명: ISO 281 표준 공식 L₁₀ = (C/P)^p
  • a_ISO 시스템 어프로치: ISO 281:2007 형식 — 단일 2D 함수 a_ISO(κ, η_c·C_u/P), cap 50, 베어링 종류별 (α, β) 분기 (radial ball/roller, thrust ball/roller).
  • 신뢰도 a₁: Weibull 분포 기반(β=1.5), 표준 테이블 일치
  • 윤활 보정: κ 구간 연속 함수(경계 매칭), Walther 점도-온도, ISO/TS 16281 참고
  • 오염도 η_c: ISO 4406 기반 (η_c=1.0~0.3)
  • 온도 보정: DIN 626 / 제조사 카탈로그 (ISO 281:2007 외 확장; tempFactor·C 적용)
  • 마찰 토크: Palmgren / SKF 단순화 (M = M₀ + M₁), ±30% 추정
  • 최소 하중 / 재급유 주기: SKF 공식 기반

⚙️ 계산 모드

  • 1990 + DIN 626 확장 (기본): a₁·a₂·a_ISO. 재료 보정 a₂(1.0~1.5) 포함. 산업계 통념.
  • 2007 Strict: a₁·a_ISO. a₂=1 강제 (재료는 C에 이미 반영된 것으로 봄).

⚠️ 한계와 알아두실 점

  • a_ISO는 ISO 281:2007 차트의 근사식입니다. 정밀 차트 디지타이징본 대비 ±15% 가능.
  • P_u는 미입력 시 베어링 종류·크기별 자동 추정(0.8~2.5%). 카탈로그 값 입력 권장.
  • ISO/TS 16281 풀버전(롤러 슬라이스별 응력 분포, 케이지·롤러 슬립 동역학)은 본 도구 범위 밖.
  • 임계 설계는 반드시 제조사 도구 또는 시험 데이터로 검증하세요.
